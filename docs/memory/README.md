# Sistema de Memoria Avanzada (Cognitive Memory)

Inspirado en LanceDB (OpenSwarm), pero adaptado a Markdown interactivo.
El Agente debe catalogar el conocimiento en estas subcarpetas durante la Fase 4 de Síntesis.

- **`beliefs/`**: Verdades fundamentales e invariantes del sistema. Cosas que no deben cambiar sin justificación mayor.
- **`strategies/`**: Métodos y patrones sobre cómo se resuelven problemas específicos en este repo (ej. "Cómo implementamos Auth").
- **`system_patterns/`**: Decisiones arquitectónicas establecidas. Convenciones de nombres, estructura de directorios, etc.

*Decay Process*: Revisa estos archivos periódicamente y elimina los que estén obsoletos.
